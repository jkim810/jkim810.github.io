ECE150: Digital Logic Design

1. Braille Implementation with LED

Digital Image of the Implemented System

Description: The project aimed to reproduce the patterns of dots - which in this case was implemented with patterns of LEDs - for numbers from 0 to 9 of the braille system. The decimal numbers were coded into binary and the expressions were simplified using the six most common logic gates: AND, OR, NAND, NOR, NOT, XOR. The system specs were set to minimize the number of gates used and to light LEDs on zeros instead of ones to break the prejudice that one means on and zero means off.

2. Traffic Light Control System

Digital Image of the Implemented System

Description: The project aimed to reproduce a traffic light system based off the one implemented in NYC. As implemented in NYC, the green and red light in N/S side was implemented longer than the E/W side as avenues are wider than streets. The following table shows the six states that traffic light system can be in.

State Table of the Traffic System

3. Elevator Control System

Digital Image of the Implemented System

Description: The project aimed to reproduce a two elevator system. The project consisted of two distinct parts: the first part aimed at producing an elevator that would store floors that it needed to stop at; the second part aimed at sorting which elevator should go to which floor. The project did not build an elevator to demonstrate this but did show how the system would work with LEDs to indicate which elevator is where.
